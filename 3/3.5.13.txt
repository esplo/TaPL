(1)

- 3.5.4: 成り立たない

最後に利用された規則が複数存在するため。

if true then t2 else t3
は E-IFTRUE と E-FUNNY1 の両規則に当てはまり、それぞれの評価が t2, t3 のいずれにもなり得るため。



- 3.5.7: 成り立つ

値に影響はない


- 3.5.8: 成り立つ

追加により、当てはまる評価規則が減り、正規形が増えることは無いため。


- 3.5.11: 成り立たない

3.5.4と同様、評価結果がt2にもt3にも成り得るため、t2 != t3 となり成り立たない。


- 3.5.12: 成り立つ

追加された規則も、項のサイズは減らすため。



(2)

- 3.5.4: 成り立たない

最後に利用された規則が複数存在するため。
例えば、
if (if true then t1 else t2) then (if true then t3 else t4) else t5
はE-IFとE-FUNNY2に当てはまる。


- 3.5.7: 成り立つ

値から値に評価されることは無いため、影響はない


- 3.5.8: 成り立つ

追加により、当てはまる評価規則が減り、正規形が増えることは無いため。


- 3.5.11: 成り立つ

E-FUNNY2は前提に基づき評価を行うだけで、既存の評価結果と一致するため


- 3.5.12: 成り立つ

追加された規則も、項のサイズは減らすため。
